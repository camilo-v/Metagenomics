Detecting Bacterial Genomes in a Metagenomic Sample Using NGS Reads.

The paper proposes a system for detecting the presence of strains of bacteria in metagenomic samples generated using whole-genome hight-throughput DNA sequencing, or Next-Generation Sequencing (NGS).

We use a nucleotide flipping technique on whole genome next generation sequencing (NGS) data to test for the pres- ence of various bacterial strains in a single metagenomic sample. We introduce artificial point mutations on the NGS reads at the nucleotide level to define a test statistic for each genome in bacterial reference database; we also use a variant of the Westfall-Young procedure to correct for multiple comparisons.
